## CRAYFIS

Эксперимент [CRAYFIS](http://crayfis.io) посвящен созданию обсерватории для изучения космического излучения ультра-высоких энергий (UHECR). Подобные частицы крайне редки (1-100 лет на км^2), поэтому, чтобы собрать достаточно статистики для их изучения, требуются либо большие времена наблюдения, либо огромная площадь покрытия детектора. Из-за трудностей в их изучении, они представляют особый интерес для астрофизиков.

Проект [CRAYFIS](http://crayfis.io) предлагает исользовать существующие телефоны, чтобы обеспечить большую площадь покрытия детектора. Камеры телефонов создаются по похожей технологии, что и некоторые детекторы элементарных частиц, поэтому частицы с относительно высокими энергиями оставляют яркий след на камере. Вместе с GPS, WiFi и наличием небольшой вычислительной мощности телефон превращается в карманную лабораторию. Таким образом телефоны добровольцев превращаются в ячейки распределенной обсерватории. Перед тем как оставить телефон в покое на ночь, владелец оставляет его камерой _вниз_ и включает специальное приложение, которое собирает данные:

> Обычный телефон днем, охотник за космическим излучением ночью.

Миллион телефонов (из миллиардов выпущеных), одновременно сканирующих небо, превращаются в обсерваторию масштабов сравнимым с самыми крупными обсерваториями для наблюдения за UHECR.

Среди задач проекта существуют несколько групп:
- алгоритмы для телефона: статистика и машинное обучение в условиях малых вычислительных мощьностей (Python, Java без програмирования для Android или iOS)
- сервер приема и обработки данных: распределенные вычисления (Scala/Java, Apache Spark, Apache Cassandra, Elastic Search etc.)
- исследовательские задачи (Python, sklearn, theano & Co. etc)

### Hot-pixel removal/calibration	

Перед получением данных камера требует каллибровки и удаления 'горячих пикселей'.

Предлагается разработать алгоритм настройки/калибровки камеры телефона для детектирования космического излучения.
Задача unsupervised learning.

Данные: внутренние, можно получить своими руками.

Необходимые навыки (один из пунктов; зависит от конкретной постановки задачи):
- Java (для разработки алгоритма на телефоне)
- Scala + Spark MLLib (для очистки данных на сервере)
- Python + scipy + sklearn (для общего тестирования и разработки алгоритмов)

Существует базовое решение.

### CRAYFIS-offline

Система приема и обработки данных с телефонов. Задача из серии "Big Data" (распределенные вычисления).
Среди подзадач:
- прием и хранение данных;
- мониторинг характеристик обсерватории;
- вычисление статистик и детектирование частиц ультра-высоких энергий.

Необходимые навыки:
- Scala + Spark
